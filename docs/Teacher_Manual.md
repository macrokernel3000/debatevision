# Teacher Manual

這份手冊是給老師備課與上課使用，不是給學生投影閱讀。

網站版手冊放在：

```text
website/pages/teacher-manual.html
```

## 上課前

1. 雙擊 `網站更新.command`。
2. 按 Enter 開啟網站。
3. 選擇今天要玩的玩法。
4. 到「本局抽選池」取消不想出現的卡。
5. 若有投影，確認學生看得到活動區，不會看到不該公布的答案。

## 通用教學節奏

每一局都可以用四步驟帶：

1. 設定：選玩法、確認卡池、調整抽取數量。
2. 抽取或選定：產生本局題目。
3. 表達與追問：學生說明，其他人質疑或補充。
4. 收束：公布答案、比較標準、投票或整理有效說法。

## 玩法七：卡片字典

卡片字典是給老師臨時組合活動用的自由玩法。

### 使用方式

1. 在左側清單查看目前所有卡池。
2. 勾選想使用的卡片類型，例如異境卡、生物卡、需求卡。
3. 切換已啟用的卡池，直接勾選本場要使用的卡。
4. 可跨卡池加入多張卡，也可以從已選清單刪除。
5. 按「儲存成這場」，系統會把已選卡牌整理成正式活動卡。

### 適合用途

- 想臨時設計新活動。
- 想測試不同卡池組合是否有趣。
- 想讓學生自己發明規則。
- 想快速取得幾張跨類別卡牌，做比較、造句、故事或推理。

卡片字典只負責自由抽卡，不會改變其他玩法的本局抽選池。

## 玩法一：異境求生

### 適合目標

- 練習用途發想。
- 練習在限制情境中說明理由。
- 練習被質疑後補充論點。

### 開始

確認異境卡，並選擇本局要抽道具或職業。主要卡可抽 1 到 6 張，適合個人或小組。

若想讓同一個異境連續挑戰不同道具或職業，可以勾選「鎖定異境」。勾選後會保留目前異境，只重抽主要卡。

### 進行

學生要說明道具或職業在異境中如何救命、突破困境或製造優勢。

### 結束

請學生回頭檢查：哪個用途最具體？哪個限制沒有處理好？

### 再來一場

重新抽出求生組合，或在抽選池取消剛剛用過的卡後再抽。

## 玩法二：存席辯護

### 適合目標

- 練習自我辯護。
- 練習比較角色價值。
- 練習反駁與淘汰投票。

### 開始

抽 1 到 6 張職業卡。可依班級人數決定抽幾張職業，先不加入異境限制。

### 進行

每位學生說服大家自己的職業最值得留下。若一次抽多張，可分組代表不同職業辯護。

### 結束

用「必要性、立即性、替代性」比較誰更重要。

### 再來一場

重新抽職業，或保留一個職業當擂主，讓新職業輪流挑戰。

## 玩法三：誰更重要

### 適合目標

- 練習建立比較標準。
- 練習不是只講喜好，而是講判準。
- 練習正反互攻。

### 開始

預設系統抽出兩張名人卡。老師也可以用上方切換按鈕改成物品卡、需求卡、概念卡、異境卡、場地卡、職業卡或生物卡，讓同一套玩法比較不同類型的詞條。

### 進行

學生各自支持一張卡，先建立比較標準，再說明為何更重要。

### 結束

觀眾投票時不能只投喜歡哪個，要說明採用哪個標準。

### 再來一場

再抽兩張卡，或保留其中一張當擂主。

## 玩法四：銷售密令

銷售密令可以單獨抽商品、單獨抽需求，也可以抽「商品 × 需求」。商品版本適合練習找賣點；需求版本適合練習理解人為什麼想買；商品加需求版本適合練習把兩者硬接起來，做成更有挑戰的銷售提案。

### 適合目標

- 練習需求分析。
- 練習目標客群。
- 練習把功能包裝成故事。

### 開始

選擇商品、需求或商品加需求。分組時可以一次抽多張。

### 進行

每組先回答：這個商品滿足什麼需求？賣給誰？怎麼包裝故事？

### 結束

同學用價格、替代品、可信度提問。

### 再來一場

抽新商品，或要求同一商品改賣給不同族群。

## 玩法五：推理解密

### 適合目標

- 練習提問。
- 練習排除法。
- 練習從線索縮小範圍。

### 開始

先選定本局要使用的詞庫，例如名人卡、場地卡、異境卡、物品卡、需求卡、職業卡或生物卡。系統會列出目前啟用的候選編號。輸入秘密編號後，即可開始活動。

### 進行

學生只能問問題，不能一開始直接猜答案。

可以回答：

- 是
- 不是
- 不一定
- 接近了

### 結束

逐一點選候選詞條。錯的會顯示「不是這個」，正確的會公布答案。

### 再來一場

按「再來一場」，重新輸入秘密編號。

## 玩法六：隱喻羅盤

### 適合目標

- 練習抽象概念的定義。
- 練習把兩個看似無關的詞建立關係。
- 練習用例子、反例與重新詮釋支撐一句話。

### 開始

系統會抽出前綴、介係、後綴，組成一句「A 關係 C」的命題，例如「自由就是運氣」。

前綴與後綴可以分別選擇要使用概念卡或需求卡。概念卡適合抽象價值與性格詞，需求卡適合動機、慾望與生活需求，例如「成功來自成就」或「愛促進正義」。

第一輪抽出後，可以分別勾選「鎖定前綴」、「鎖定介係」、「鎖定後綴」。鎖定後下一輪只會重抽沒有鎖定的位置，適合固定一個詞反覆練習不同組合。

### 進行

學生先定義兩個概念，再說明中間的關係為什麼可以成立。重點不是證明字面完全正確，而是找到一個有說服力的角度。

### 結束

同學可以提出反例，或嘗試改寫中間的關係詞，看看命題是否變得更準確。

### 再來一場

重新抽一組命題，或用鎖定功能保留前綴、介係或後綴，只重抽其他位置。也可以改變前綴或後綴使用的卡池，讓同一個關係詞連接不同類型的詞。

## 常見調整

### 改玩法文字

修改：

```text
data/content/玩法文案.csv
```

### 改卡牌內容

修改：

```text
data/cards/*.csv
```

隱喻羅盤目前主要修改：

```text
data/cards/概念卡.csv
data/cards/關係卡.csv
```

### 改抽選池小符號

修改卡牌 CSV 的：

```text
抽選池圖示
```

可以填 `✚`、`♡`、`文`、`🧠` 或其他短符號。

### 改抽出卡牌的大圖示

修改卡牌 CSV 的：

```text
卡牌圖示
```

圖片放在：

```text
assets/icons/{牌組ID}/
```

### 改玩法背景

修改：

```text
data/content/玩法文案.csv
```

欄位填：

```text
玩法背景
```

圖片通常放在：

```text
assets/backgrounds/modes/
```

## 上課提醒

- 投影前先測一局。
- 不想抽到的卡先取消勾選。
- 已抽過的卡會自動從本局抽選池取消。
- 如果學生程度較低，先要求「說一個用途」即可。
- 如果學生程度較高，加入質疑、比較標準與投票。
